Gesture-Controlled Turtle-Bot

Developed a turtle-bot controlled by hand gestures using accelerometer and gyroscope data. The bot responds to gestures, allowing remote control without a joystick or physical touch.

Smart Incubator Controller

Designed a controller that monitors incubation status and sends sensor diagnostics via SMS. This project involved hardware integration and GSM communication protocols.

Radar Obstacle Detection

Using an ESP32, servos, and ultrasonic sensors, this radar project detects obstacles and plots their positions. A web interface was created to visualize real-time data.
